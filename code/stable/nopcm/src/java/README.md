## Making Examples 
 How to Run the Program:
In your terminal command line, enter the same directory as this README file. Then enter the following line:
```
make run RUNARGS=input.txt
```

How to Build the Program:
In your terminal command line, enter the same directory as this README file. Then enter the following line:
```
make build
```
