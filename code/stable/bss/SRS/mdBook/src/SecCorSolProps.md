# Properties of a Correct Solution {#Sec:CorSolProps}

The total mechanical energy of the system must remain constant over time (up to numerical tolerance), since the system is isolated ([A:isolated](./SecAssumps.md#isolated)) and masses are constant ([A:constantMass](./SecAssumps.md#constantMass)).
