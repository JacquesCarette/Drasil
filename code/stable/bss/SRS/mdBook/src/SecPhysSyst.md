# Physical System Description {#Sec:PhysSyst}

The physical system of BSS, as shown in [Fig:bssPhysSys](./SecPhysSyst.md#Figure:bssPhysSys), includes the following elements:

PS1: The first star with mass \\({m\_{1}}\\).

PS2: The second star with mass \\({m\_{2}}\\).

PS3: The gravitational interaction between the two stars.
