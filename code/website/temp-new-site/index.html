This webpage is designed to contain the most up to date case study artifacts and package dependency graphs from the Drasil repository. Each of the case studies contain their own generated PDF and HTML reports, and in some cases, their own generated code.


Case Studies

  • Chipmunk - describes the modeling of an open source 2D rigid body physics library used for games.
  • GlassBR - predicts whether a given glass slab is likely to resist a specified blast.
  • NoPCM - describe the modelling of a solar water heating system without phase change material.
  • Projectile - describes the motion of a projectile object in free space.
  • SSP - describes the requirements of a slope stability analysis program.
  • SWHS - describe the modelling of a solar water heating system with phase change material.
  • Template - an empty template document.
  • Tiny - describes heat transfer coefficients related to clad
